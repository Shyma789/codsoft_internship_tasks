# codsoft_task1
Titanic Survival Prediction using Python and Machine Learning.  This project analyzes the Titanic dataset and predicts passenger survival using classification techniques.  It also includes data visualization using Matplotlib and Seaborn to identify patterns such as survival by gender, class, and age.
# Titanic Survival Prediction 🚢

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques.

## 📊 Features
- Data preprocessing and cleaning
- Survival prediction using classification
- Data visualization using Matplotlib and Seaborn

## 📈 Visualizations
- Survival count
- Survival by gender
- Survival by passenger class
- Age distribution
- Correlation heatmap

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
