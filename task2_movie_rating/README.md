# 🎬 Movie Rating Prediction

## 📌 Project Overview
This project focuses on predicting movie ratings based on various features such as genre, director, and audience engagement. It applies data analysis and machine learning techniques to estimate how a movie might be rated by users or critics.

---

## 🎯 Objectives
- Analyze historical movie data
- Perform data preprocessing and cleaning
- Build a regression model for rating prediction
- Visualize patterns affecting movie ratings

---

## 📊 Features
- Data cleaning and handling missing values
- Feature analysis (genre, director, votes, etc.)
- Movie rating prediction using regression techniques
- Advanced data visualization using Matplotlib and Seaborn

---

## 📈 Visualizations Included
- Rating Distribution Curve
- Average Rating by Genre
- Director-wise Rating Analysis
- Votes vs Rating Relationship

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📁 Dataset
- IMDb Movie Dataset (`imdbmovieindia.csv`)
- Contains information such as movie title, genre, director, ratings, and votes

---

## ▶️ How to Run the Project

1. Open terminal
2. Navigate to the project folder:cd task2_movie_rating
3. Install required libraries:pip install -r requirements.txt
4. Run the Python file:python movie_rating.py
---


## ✅ Output
- Predicts movie ratings based on input features
- Displays multiple graphs for data analysis and insights

---

## 📌 Key Insights
- Genre plays a significant role in movie ratings  
- Popular directors tend to produce higher-rated movies  
- Movies with higher votes often show consistent rating trends  
- Audience engagement influences rating prediction
